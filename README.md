# essays-and-meditations

Personal bilingual (English/Persian) blog powered by **GitHub Pages + Jekyll**.

## Structure

- `index.md` — homepage with language links
- `en/` — English pages
- `fa/` — Persian pages
- `_essays_en/` — English essay markdown files
- `_essays_fa/` — Persian essay markdown files

Add new markdown files to `_essays_en` or `_essays_fa` and they will be listed automatically on the language pages.
